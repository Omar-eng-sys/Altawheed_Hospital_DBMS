# 🏥 Al-Tawheed Hospital DBMS

Welcome to Phase 2 (SQL Implementation). Please follow this workflow strictly.

## 🚀 Git Workflow

**1. Setup (First time only):**
```bash
git clone [https://github.com/Omar-eng-sys/Altawheed_Hospital_DBMS.git](https://github.com/Omar-eng-sys/Altawheed_Hospital_DBMS.git)
```

**2. Update (Every time you start):**
```bash
cd Altawheed_Hospital_DBMS
git pull
```

**3. Write & Test Code (The Golden Rule):**
* **NEVER** type new code directly into `Hospital_Management_System.sql`.
* Test your code in the SQL Command Line first.
* Only when it says `Table created`, **COPY & PASTE** it into your assigned section.

**4. Upload (When finished):**
```bash
git add .
git commit -m "Short description of what you did"
git push
```
